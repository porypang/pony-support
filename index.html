Pony 销售助手

iOSv1.0.1本地存储

你的私人销售管理工具。iPhone 就能用的客户管理、报价单、产品库 App。

📋 App 功能

  • 数据看板 — 客户总数、报价数量、产品库规模一目了然
  • 客户管理 — 联系人、公司、电话、邮箱、地址、潜力评估
  • 报价单管理 — 多产品报价，自动 PDF 导出分享
  • 产品库管理 — JSON 文件导入/导出，1000+ 产品随时可用

❓ 常见问题

Q: 我的数据会上传到服务器吗？

不会。Pony 销售助手所有数据都存储在您的 iPhone 本地，不上传任何服务器。您可以随时清除 App 数据，删除 App 后所有数据自动清除。

Q: 需要联网才能使用吗？

不需要。App 全部功能在离线状态下可用，包括生成 PDF 报价单、查看客户资料、产品库搜索等。

Q: 如何备份我的数据？

目前数据保存在设备本地。建议：1) 定期通过 iCloud/iTunes 备份您的 iPhone；2) 产品库可通过"导出 JSON"功能手动备份到其他位置。

Q: 产品库 JSON 格式是什么？

支持两种格式：
格式 A: {"products": [{"itemNo": "型号", "name": "名称", "price": 价格, "category": "类别"}], "total": 数量}
格式 B: [{"itemNo": "型号", "name": "名称", "price": 价格, "category": "类别"}]

Q: 怎么导入产品库？

Dashboard → 产品数据维护 → 点击"导入 JSON 产品库" → 选择您的 JSON 文件即可。系统会自动跳过重复的型号。

Q: 支持哪些 iOS 版本？

需要 iOS 17.0 或更高版本。适配 iPhone 和 iPad。
